# PythonPowderGame V1

**1** - Sand
**2** - Stone
**3** - Water
**0** - Empty

## Adding a new material
- Define a constant in config.py, e.g "SAND = 1"
- Implement it's behaviour in materials.py 
- Add update logic to update() in grid.py
- Assign a color in renderer.py 
- Map to hotkey in main.py for placement
